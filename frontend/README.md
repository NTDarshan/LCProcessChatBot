# Frontend

Angular 21 starter application configured with zoneless change detection and Angular Material.

## Included

- Angular 21.2.x
- zoneless setup with no `zone.js`
- Angular Material 21
- standalone components
- SCSS and routing

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm start
```

Open `http://localhost:4200/`.

## Useful Commands

Build:

```bash
npm run build
```

Test:

```bash
npm test
```

Generate code:

```bash
npx ng generate component feature-name
```
